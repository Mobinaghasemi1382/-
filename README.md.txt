# 🛍️ فروشگاه اینترنتی عطر و ادکلن

!پیشنمایش فروشگاه

## ✨ ویژگی‌های اصلی
- نمایش محصولات با فیلترهای پیشرفته (جنسیت، برند، قیمت)
- سبد خرید ساده و کاربردی
- سیستم ارزیابی و نظرات
- طراحی ریسپانسیو برای موبایل و دسکتاپ

## 🚀 راه‌اندازی پروژه

### پیش‌نیازها
- Node.js نسخه 16+
- npm یا yarn

### نصب و اجرا
```bash
# 1. کپی کردن پروژه
git clone https://github.com/username/perfume-shop.git

# 2. نصب بسته‌ها
npm install

# 3. اجرای برنامه
npm start
